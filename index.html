웹 프로그래밍 '홈페이지 작성' 과제 체크리스트

본 과제에서 주차에 있는 내용을 모두 적용하려했으며, 프로필 사이트 제작을 해보았습니다.

  • 1주차(개념)
    1. Html 무엇인가
    2. 웹 표준이란 무엇인가
    3. 웹 표준을 사용하는 이유
    4. HTML5와 CSS3를 공부해야하는 이유
  • 2주차
    1. html / head / body 태그
    2. hn / br / p / hr 태그
    3. blockquote 인용문 넣기 / pre 입력한 그대로 표시
  • 3주차
    1. storng 굵게 표시 / b 태그를 잘 안쓰는 이유 / em 이탤릭체 / q 인용표시 ""로 표기 / mark 형광펜
    2. span 줄 안에서 묶기 / div 단락으로 묶기
    3. ul 순서없는 목록 대괄호 태그 / ol 순서있는 목록 대괄호 태그 / li 리스트 태그
    4. dl 목록 대괄호 dt 줄목록 dd 줄글목록
  • 4주차
    1. table 테이블 태그
    2. tr ~ /tr : 행 td ~ /td : 셀, th ~ /th 제목 셀
    3. colspan, rowspan : 행 또는 열 합치기, 몇 칸할 것인지 정해줌
    4. colgroup ~ /colgroup : 열의 스타일 표시
    5. figcaption : 표 제목 (속성 위치에 따라 표의 위나 아래에 제목 표시)
    6. thead / tbody / tfoot : 표 구조 정의하기
    7. img 이미지 태그 / src 경로 속성 / width, height : 이미지크기 조정 속성
    8. figure figcaption : 이미지에 설명글 붙이기 (figure 안에 img 태그하고 img 위 figcaption img 아래 figcaption이 다름)
