웹 프로그래밍 '홈페이지 작성' 과제 체크리스트

line-through 는 홈페이지에서 사용한 부분을 의미합니다. ( 개념 제외 )

  • 1주차(개념)
    1. Html 무엇인가
    2. 웹 표준이란 무엇인가
    3. 웹 표준을 사용하는 이유
    4. HTML5와 CSS3를 공부해야하는 이유
  • 2주차
    1. html / head / body 태그
    2. hn / br / p / hr 태그
    3. blockquote 인용문 넣기 / pre 입력한 그대로 표시
  • 3주차
    1. storng 굵게 표시 / b 태그를 잘 안쓰는 이유 / em 이탤릭체 / q 인용표시 ""로 표기 / mark 형광펜
    2. span 줄 안에서 묶기 / div 단락으로 묶기
    3. ul 순서없는 목록 대괄호 태그 / ol 순서있는 목록 대괄호 태그 / li 리스트 태그
    4. dl 목록 대괄호 dt 줄목록 dd 줄글목록
  • 4주차
    1. table 테이블 태그
    2. tr ~ /tr : 행 td ~ /td : 셀, th ~ /th 제목 셀
    3. colspan, rowspan : 행 또는 열 합치기, 몇 칸할 것인지 정해줌
    4. colgroup ~ /colgroup : 열의 스타일 표시
    5. figcaption : 표 제목 (속성 위치에 따라 표의 위나 아래에 제목 표시)
    6. thead / tbody / tfoot : 표 구조 정의하기
    7. img 이미지 태그 / src 경로 속성 / width, height : 이미지크기 조정 속성
    8. figure figcaption : 이미지에 설명글 붙이기 (figure 안에 img 태그하고 img 위 figcaption img 아래 figcaption이 다름)
  • 5주차
    1. a 하이퍼링크태그 - target 속성 : 링크를 새 탭에서 열기
    2. 한 페이지 안에서 점프하기 - 앵커 id와 #을 이용하여 표현
    3. area 태그, usemap 속성( 사용자가 웹 사이트로 정보를 보낼 수 있는 요소 : 폼 )
  • 6주차
    1. input 태그 / type / user-name / size
  • 7주차
    1. input에 대한 다양한 폼 / CSS 기초 버튼 / 퍼센트 그래프 표현
    2. 내부 스타일 : 모든 스타일 정보는 head 안에 정의 style 사이에 작성
    3. 외부 스타일 : css 파일로 저장하여 link태그를 이용하여 사용, style.css / link rel="stylesheet" href="css/style.css"
    4. 직접적으로 인라인으로 표시하기도 한다. 전체 선택자 *{}
    5. 클래스와 id 선택자 / 문서 안에서 여러번 반복할 스타일이면 class / 문서 안에서 한 번만 사용한다면 id 선택자 # 다음에 id 이름 지정
    6. 그룹으로 묶어서 지정하는 그룹 선택자도 있음. 태그, 태그{}
    7. 캐스캐이딩 스타일 시트( CSS ) : 위에서 아래로 흐른다의 의미로 우선순위 / 상속에 따라 위에서 아래로 전달됨.
  • 8주차
    1. 원칙 1: 스타일 우선 순위 / 원칙 2: 스타일 상속
    2. 다른 브라우저를 위해 브라우저 접두사를 붙여 사용하기도 함
    3. 텍스트 관련 스타일 : 직접 웹 폰트 업로드해서 사용하기 / font-size / font-weight / font-variant /
    4. font-style / font / color / text-decoration(none, underline, overline, line-through ) / text-transform (대소, 소대 전환)
    5. text-shadow 속성 / white-space 속성 ( 자간 )
  • 9주차
    1. direction 속성 : 텍스트를 쓰는 방향 지정
    2. text-align / text-indent / line-height / text-overflow / list-style-type
    3. list-style-image / list-style-position ( inside, outside )

    4. background-color / background-clip / background-image / background-repeat / background-size / background-position
    5. background-origin / background-attachment
    6. background

    7. 그라데이션 속성, 그라데이션 반복
    8. background: linear-gradient(to right bottom, blue, white)

    9. (CSS 박스 모델) 블록 레벨 요소 - 요소를 삽입했을 때 혼자 한 줄을 차지하는 요소 div p
    10. (CSS 박스 모델) 인라인 레벨 요소 - 화면에 표시되는 만큼만 영역을 차지하고 나머지 공간에는 다른 요소가 올 수 있음. img strong

    11. 박스 모델 - 실제 콘텐츠 영역, 패딩, 박스의 테두리, 그리고 마진 등의 요소로 구성됨.
    12. width, height ( px 고정너비, % 화면에 따른 가변 너비 )
    13. display 속성 : none | contents | block | inline | inline-block | table | table-cell
    14. Border-style / Border-width / Border-color / Border-radius
    15. Box-shadow : 요소에 그림자 효과 내기
    16. Margin 속성 : 마진을 이용하면 요소 간의 간격 조절 가능 1개라면 모두 / 2개라면 위아래 / 좌우 마진 값 3개라면 위 / 좌우 / 아래 / 4개 라면 위 / 오른 / 아래 / 왼 순으로 적용
    17. Padding 속성 : 콘텐츠 영역과 테두리 사이의 여백( 테두리 왼쪽 여백 )
  • 10주차
    1. css 포지셔닝 : float 속성 / clear 속성 / position 속성(static, relative, absolute, fixed) / visibility 속성 / z-index 속성 요소 쌓는 순서 정하기
  • 11주차
    1. 문단 나누기 : column-width : 문단 너비 column-count 단의 개수 break-before, break-after 특정 요소의 앞이나 뒤에 다단 위치 지정 ( 단 새로시작 )
    2. column-span ( 여러 단을 하나로 합치기 )
  • 12주차
    1. 내용 없음
  • 13주차
    1. 내용 없음
